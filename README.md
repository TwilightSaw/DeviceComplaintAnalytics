# Data Analytic Project

## This is a showcase of Excel, Power BI and SQL usage.

## How to recreate the dataset

> [!IMPORTANT]
> After downloading the project, you should execute db_creation_queries.sql, then insertion_queries.sql, then dataset.sql to achieve the final dataset.
> Open .pbix Power BI file.
